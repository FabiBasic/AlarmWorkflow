AlarmWorkflow v0.9.8.0 Release Notes
====================================

Änderungen seit v0.9.7.0
------------------------

Generelles

  * [AlarmSourceJob] Seiten werden auf die Seitengröße skaliert, sodass keine Inhalte mehr abgeschnitten werden.
  * [Website] Es können neue Alarmseiten erstellt werden. Diese können unter C:\inetpub\wwwroot\Areas\Display\Views\Alarm erstellt werden. Der Dateiname (ohne cshtml) dient zur Identifizierung. http://localhost/Display/Alarm/Index/**NAME** kann als Aufruf verwendet werden.
  * [FaxAlarmSource] Einige Einstellungen werden nun auch dynamisch geändert.
